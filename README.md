## Ptaha – Corporate Website with Custom CMS

Overview

Corporate website built with Laravel 12 and a custom CMS powered by Filament and Livewire.

The project demonstrates structured backend management combined with a modern frontend built using Blade and TailwindCSS.

⸻

## Key Features
•	Custom CMS for content management  
•	Dynamic page structure  
•	Admin panel built with Filament  
•	Modern UI with TailwindCSS  
•	Clean separation between content and layout
